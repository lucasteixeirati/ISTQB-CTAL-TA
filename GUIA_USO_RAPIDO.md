# GUIA DE USO RÁPIDO - ISTQB CTAL-TA

---

## DICAS RÁPIDAS

- Use sempre os arquivos de capítulo para questões atualizadas
- Flashcards: revise diariamente para melhor retenção
- Dashboard: acompanhe seu progresso semanalmente
- Simule condições reais no modo exame

---

## SOLUÇÃO DE PROBLEMAS

- Erro de módulo: verifique se está na pasta correta e se os arquivos existem
- Arquivo de progresso não encontrado: será criado automaticamente
- Simulado não gera questões: teste o banco com `python banco_questoes_completo.py`

---

## COMANDOS RÁPIDOS

- Banco de questões:
  ```bash
  python banco_questoes_completo.py
  ```
- Gerar simulado:
  ```bash
  python gerador_simulados_v3.py
  ```
- Dashboard de progresso:
  ```bash
  python tracking_progresso_v2.py
  ```
- Flashcards do dia:
  ```bash
  python sistema_revisao.py
  ```
