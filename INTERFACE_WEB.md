# 🌐 INTERFACE WEB - ISTQB CTAL-TA v3.0

## Instalação

1. Instale as dependências:
```bash
cd c:\Projetos\ISTQB-CTAL-TA
pip install -r requirements.txt
```
2. Inicie o servidor:
```bash
python app.py
```
3. Acesse em: http://localhost:5000

---

## Funcionalidades

- Página inicial moderna e responsiva
- Simulado interativo (cronômetro, navegação, resultado)
- Upload e geração de questões
- Dashboard de progresso
- Flashcards digitais
